\doxysubsection{Detailed Description}
An abstract class that all sniff unit tests must extend.

\mbox{\hyperlink{class_a}{A}} sniff unit test checks a .inc file for expected violations of a single coding standard. Expected errors and warnings that are not found, or warnings and errors that are not expected, are considered test failures.
