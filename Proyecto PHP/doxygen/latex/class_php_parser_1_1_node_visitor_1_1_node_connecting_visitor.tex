\doxysubsection{Detailed Description}
Visitor that connects a child node to its parent node as well as its sibling nodes.

On the child node, the parent node can be accessed through {\ttfamily \$node-\/\texorpdfstring{$>$}{>}get\+Attribute(\textquotesingle{}parent\textquotesingle{})}, the previous node can be accessed through {\ttfamily \$node-\/\texorpdfstring{$>$}{>}get\+Attribute(\textquotesingle{}previous\textquotesingle{})}, and the next node can be accessed through {\ttfamily \$node-\/\texorpdfstring{$>$}{>}get\+Attribute(\textquotesingle{}next\textquotesingle{})}.
