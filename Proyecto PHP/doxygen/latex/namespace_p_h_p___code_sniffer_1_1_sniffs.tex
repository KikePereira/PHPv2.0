\mbox{\hyperlink{class_a}{A}} class to find T\+\_\+\+VARIABLE tokens.

This class can distinguish between normal T\+\_\+\+VARIABLE tokens, and those tokens that represent class members. If a class member is encountered, then the process\+Member\+Var method is called so the extending class can process it. If the token is found to be a normal T\+\_\+\+VARIABLE token, then process\+Variable is called.
