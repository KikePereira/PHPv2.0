\doxysubsection{Detailed Description}
Wraps Exceptions thrown by code under test.

Re-\/instantiates Exceptions thrown by user-\/space code to retain their original class names, properties, and stack traces (but without arguments).

Unlike \mbox{\hyperlink{namespace_p_h_p_unit}{PHPUnit}}\textbackslash{}\+Framework\textbackslash{}\+Exception, the complete stack of previous Exceptions is processed.
