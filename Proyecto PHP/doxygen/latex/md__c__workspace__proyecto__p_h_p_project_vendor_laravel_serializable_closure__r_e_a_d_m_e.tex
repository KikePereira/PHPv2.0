\begin{quote}
This project is a fork of the excellent \href{https://github.com/opis/closure}{\texttt{ opis/closure\+: 3.\+x}} package. At Laravel, we decided to fork this package as the upcoming version \href{https://github.com/opis/closure}{\texttt{ 4.\+x}} is a complete rewrite on top of the \href{https://www.php.net/manual/en/book.ffi.php}{\texttt{ FFI extension}}. As Laravel is a web framework, and FFI is not enabled by default in web requests, this fork allows us to keep using the {\ttfamily 3.\+x} series while adding support for new PHP versions. \end{quote}
Laravel Serializable Closure provides an easy and secure way to {\bfseries{serialize closures in PHP}}.\hypertarget{md__c__workspace__proyecto__p_h_p_project_vendor_laravel_serializable_closure__r_e_a_d_m_e_autotoc_md99}{}\doxysection{Official Documentation}\label{md__c__workspace__proyecto__p_h_p_project_vendor_laravel_serializable_closure__r_e_a_d_m_e_autotoc_md99}
