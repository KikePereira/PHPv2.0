\doxysubsection{Detailed Description}
Checks if casting an expression to \mbox{\hyperlink{class_twig_1_1_node_1_1_node_a7516ca30af0db3cdbf9a7739b48ce91d}{\+\_\+\+\_\+to\+String()}} is allowed by the sandbox.

For instance, when there is a simple Print statement, like \{\{ article \}\}, and if the sandbox is enabled, we need to check that the \mbox{\hyperlink{class_twig_1_1_node_1_1_node_a7516ca30af0db3cdbf9a7739b48ce91d}{\+\_\+\+\_\+to\+String()}} method is allowed if \textquotesingle{}article\textquotesingle{} is an object. The same goes for \{\{ article$\vert$upper \}\} or \{\{ random(article) \}\}
