\doxysubsection{Detailed Description}
Provides \mbox{\hyperlink{namespace_d_i}{DI}} definitions by reading annotations such as @ Inject and @ var annotations.

Uses \mbox{\hyperlink{interface_d_i_1_1_definition_1_1_source_1_1_autowiring}{Autowiring}}, Doctrine\textquotesingle{}s Annotations and regex docblock parsing. This source automatically includes the reflection source.
