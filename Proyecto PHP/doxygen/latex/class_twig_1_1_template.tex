\doxysubsection{Detailed Description}
Default base class for compiled templates.

This class is an implementation detail of how template compilation currently works, which might change. It should never be used directly. Use \$twig-\/\texorpdfstring{$>$}{>}load() instead, which returns an instance of \textbackslash{}\+Twig\textbackslash{}\+Template\+Wrapper.
