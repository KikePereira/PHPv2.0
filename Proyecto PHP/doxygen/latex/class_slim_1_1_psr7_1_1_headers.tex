{\footnotesize\ttfamily validate\+Header (\begin{DoxyParamCaption}\item[{}]{\$name,  }\item[{}]{\$value }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [protected]}}

Make sure the header complies with RFC 7230.

\mbox{\hyperlink{class_slim_1_1_psr7_1_1_header}{Header}} names must be a non-\/empty string consisting of token characters.

\mbox{\hyperlink{class_slim_1_1_psr7_1_1_header}{Header}} values must be strings consisting of visible characters with all optional leading and trailing whitespace stripped. This method will always strip such optional whitespace. Note that the method does not allow folding whitespace within the values as this was deprecated for almost all instances by the RFC.
