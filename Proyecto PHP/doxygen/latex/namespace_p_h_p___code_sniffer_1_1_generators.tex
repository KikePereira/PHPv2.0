\doxysubsection{Detailed Description}
The base class for all \mbox{\hyperlink{namespace_p_h_p___code_sniffer}{PHP\+\_\+\+Code\+Sniffer}} documentation generators.

Documentation generators are used to print documentation about code sniffs in a standard.
