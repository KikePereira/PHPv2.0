\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_a}{A}} dummy file represents a chunk of text that does not have a file system location.

Dummy files can also represent a changed (but not saved) version of a file and so can have a file path either set manually, or set by putting phpcs\+\_\+input\+\_\+file\+: /path/to/file as the first line of the file contents.
