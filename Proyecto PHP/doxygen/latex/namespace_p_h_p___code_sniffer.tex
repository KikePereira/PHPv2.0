\doxysubsection{Detailed Description}
Autoloads files for \mbox{\hyperlink{namespace_p_h_p___code_sniffer}{PHP\+\_\+\+Code\+Sniffer}} and tracks what has been loaded.

Due to different namespaces being used for custom coding standards, the autoloader keeps track of what class is loaded after a file is included, even if the file is ultimately included by another autoloader (such as composer).

This allows \mbox{\hyperlink{namespace_p_h_p___code_sniffer}{PHP\+\_\+\+Code\+Sniffer}} to request the class name after loading a class when it only knows the filename, without having to parse the file to find it.
