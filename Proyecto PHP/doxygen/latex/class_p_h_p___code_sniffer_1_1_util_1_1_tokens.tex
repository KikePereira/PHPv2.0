{\footnotesize\ttfamily static get\+Highest\+Weighted\+Token (\begin{DoxyParamCaption}\item[{array}]{\$tokens }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}

Returns the highest weighted token type.

\mbox{\hyperlink{class_p_h_p___code_sniffer_1_1_util_1_1_tokens}{Tokens}} are weighted by their approximate frequency of appearance in code
\begin{DoxyItemize}
\item the less frequently they appear in the code, the higher the weighting. For example T\+\_\+\+CLASS tokens appear very infrequently in a file, and therefore have a high weighting.
\end{DoxyItemize}

Returns false if there are no weightings for any of the specified tokens.
