\doxysubsection{Detailed Description}
Simply returns all the values of the \$provided\+Parameters array that are indexed by the parameter position (i.\+e. a number).

E.\+g. `-\/\texorpdfstring{$>$}{>}call(\$callable, \mbox{[}\textquotesingle{}foo', \textquotesingle{}bar\textquotesingle{}\mbox{]}){\ttfamily will simply resolve the parameters to}\mbox{[}\textquotesingle{}foo\textquotesingle{}, \textquotesingle{}bar\textquotesingle{}\mbox{]}\`{}.

Parameters that are not indexed by a number (i.\+e. parameter position) will be ignored.
