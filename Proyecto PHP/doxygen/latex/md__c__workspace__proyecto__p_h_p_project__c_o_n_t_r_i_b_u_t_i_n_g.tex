\begin{DoxyEnumerate}
\item Fork the \mbox{\hyperlink{namespace_slim}{Slim}} Skeleton repository
\item Create a new branch for each feature or improvement
\item Send a pull request from each feature branch to the {\bfseries{4.\+x}} branch
\end{DoxyEnumerate}

It is very important to separate new features or improvements into separate feature branches, and to send a pull request for each branch. This allows us to review and pull in new features or improvements individually.\hypertarget{md__c__workspace__proyecto__p_h_p_project__c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md2}{}\doxysection{Style Guide}\label{md__c__workspace__proyecto__p_h_p_project__c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md2}
All pull requests must adhere to the \href{https://github.com/php-fig/fig-standards/blob/master/accepted/PSR-12-extended-coding-style-guide.md}{\texttt{ PSR-\/12 standard}}.
