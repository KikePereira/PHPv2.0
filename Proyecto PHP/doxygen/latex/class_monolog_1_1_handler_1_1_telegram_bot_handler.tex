\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_monolog_1_1_handler_1_1_handler}{Handler}} send logs to Telegram using Telegram Bot API.

How to use\+: 1) Create telegram bot with \href{https://telegram.me/BotFather}{\texttt{ https\+://telegram.\+me/\+Bot\+Father}} 2) Create a telegram channel where logs will be recorded. 3) Add created bot from step 1 to the created channel from step 2.

Use telegram bot API key from step 1 and channel name with \textquotesingle{}@\textquotesingle{} prefix from step 2 to create instance of \mbox{\hyperlink{class_monolog_1_1_handler_1_1_telegram_bot_handler}{Telegram\+Bot\+Handler}}
