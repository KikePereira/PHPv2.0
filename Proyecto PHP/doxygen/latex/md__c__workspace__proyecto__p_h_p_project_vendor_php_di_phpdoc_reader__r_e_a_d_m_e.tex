It supports namespaced class names with the same resolution rules as PHP\+:


\begin{DoxyItemize}
\item fully qualified name (starting with {\ttfamily \textbackslash{}})
\item imported class name (eg. {\ttfamily use My\textbackslash{}Cache\textbackslash{}Backend;})
\item relative class name (from the current namespace, like {\ttfamily Sub\+Namespace\textbackslash{}\mbox{\hyperlink{class_my_class}{My\+Class}}})
\item aliased class name (eg. {\ttfamily use My\textbackslash{}Cache\textbackslash{}Backend as \mbox{\hyperlink{class_foo_bar}{Foo\+Bar}};})
\end{DoxyItemize}
