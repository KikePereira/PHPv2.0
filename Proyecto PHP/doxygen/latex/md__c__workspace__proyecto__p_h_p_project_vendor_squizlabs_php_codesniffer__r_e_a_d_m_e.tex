\mbox{\hyperlink{namespace_p_h_p___code_sniffer}{PHP\+\_\+\+Code\+Sniffer}} uses a {\ttfamily MAJOR.\+MINOR.\+PATCH} version number format.

The {\ttfamily MAJOR} version is incremented when\+:
\begin{DoxyItemize}
\item backwards-\/incompatible changes are made to how the {\ttfamily phpcs} or {\ttfamily phpcbf} commands are used, or
\item backwards-\/incompatible changes are made to the {\ttfamily ruleset.\+xml} format, or
\item backwards-\/incompatible changes are made to the API used by sniff developers, or
\item custom \mbox{\hyperlink{namespace_p_h_p___code_sniffer}{PHP\+\_\+\+Code\+Sniffer}} token types are removed, or
\item existing sniffs are removed from \mbox{\hyperlink{namespace_p_h_p___code_sniffer}{PHP\+\_\+\+Code\+Sniffer}} entirely
\end{DoxyItemize}

The {\ttfamily MINOR} version is incremented when\+:
\begin{DoxyItemize}
\item new backwards-\/compatible features are added to the {\ttfamily phpcs} and {\ttfamily phpcbf} commands, or
\item backwards-\/compatible changes are made to the {\ttfamily ruleset.\+xml} format, or
\item backwards-\/compatible changes are made to the API used by sniff developers, or
\item new sniffs are added to an included standard, or
\item existing sniffs are removed from an included standard
\end{DoxyItemize}

\begin{quote}
NOTE\+: Backwards-\/compatible changes to the API used by sniff developers will allow an existing sniff to continue running without producing fatal errors but may not result in the sniff reporting the same errors as it did previously without changes being required. \end{quote}
The {\ttfamily PATCH} version is incremented when\+:
\begin{DoxyItemize}
\item backwards-\/compatible bug fixes are made
\end{DoxyItemize}

\begin{quote}
NOTE\+: As \mbox{\hyperlink{namespace_p_h_p___code_sniffer}{PHP\+\_\+\+Code\+Sniffer}} exists to report and fix issues, most bugs are the result of coding standard errors being incorrectly reported or coding standard errors not being reported when they should be. This means that the messages produced by \mbox{\hyperlink{namespace_p_h_p___code_sniffer}{PHP\+\_\+\+Code\+Sniffer}}, and the fixes it makes, are likely to be different between PATCH versions. \end{quote}
