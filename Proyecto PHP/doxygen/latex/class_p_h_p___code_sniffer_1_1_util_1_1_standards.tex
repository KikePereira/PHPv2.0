Get the details of all coding standards installed.

Coding standards are directories located in the Code\+Sniffer/\+Standards directory. Valid coding standards include a \mbox{\hyperlink{namespace_p_h_p___code_sniffer_1_1_sniffs}{Sniffs}} subdirectory.

The details returned for each standard are\+:
\begin{DoxyItemize}
\item path\+: the path to the coding standard\textquotesingle{}s main directory
\item name\+: the name of the coding standard, as sourced from the ruleset.\+xml file
\item namespace\+: the namespace used by the coding standard, as sourced from the ruleset.\+xml file
\end{DoxyItemize}

If you only need the paths to the installed standards, use \mbox{\hyperlink{class_p_h_p___code_sniffer_1_1_util_1_1_standards_abc45469a9fc5542cff00dde6b7342dc7}{get\+Installed\+Standard\+Paths()}} instead as it performs less work to retrieve coding standard names.
