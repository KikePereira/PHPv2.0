\mbox{\hyperlink{namespace_p_h_p_unit}{PHPUnit}} is a programmer-\/oriented testing framework for PHP. It is an instance of the x\+Unit architecture for unit testing frameworks.

\href{https://packagist.org/packages/phpunit/phpunit}{\texttt{ }} \href{https://php.net/}{\texttt{ }} \href{https://phpunit.de/build-status.html}{\texttt{ }} \href{https://shepherd.dev/github/sebastianbergmann/phpunit}{\texttt{ }}\hypertarget{md__c__workspace__proyecto__p_h_p_project_vendor_phpunit_phpunit__r_e_a_d_m_e_autotoc_md813}{}\doxysubsection{Installation}\label{md__c__workspace__proyecto__p_h_p_project_vendor_phpunit_phpunit__r_e_a_d_m_e_autotoc_md813}
We distribute a \href{https://php.net/phar}{\texttt{ PHP Archive (PHAR)}} that has all required (as well as some optional) dependencies of \mbox{\hyperlink{namespace_p_h_p_unit}{PHPUnit}} bundled in a single file\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\$ wget https://phar.phpunit.de/phpunit-\/X.Y.phar}
\DoxyCodeLine{}
\DoxyCodeLine{\$ php phpunit-\/X.Y.phar -\/-\/version}

\end{DoxyCode}


Please replace {\ttfamily X.\+Y} with the version of \mbox{\hyperlink{namespace_p_h_p_unit}{PHPUnit}} you are interested in.

Alternatively, you may use \href{https://getcomposer.org/}{\texttt{ Composer}} to download and install \mbox{\hyperlink{namespace_p_h_p_unit}{PHPUnit}} as well as its dependencies.
