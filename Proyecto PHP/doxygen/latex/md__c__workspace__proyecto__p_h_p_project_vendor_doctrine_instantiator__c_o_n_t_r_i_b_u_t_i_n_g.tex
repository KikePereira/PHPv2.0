To install the project and run the tests, you need to clone it first\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\$ git clone git://github.com/doctrine/instantiator.git}

\end{DoxyCode}


You will then need to run a composer installation\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\$ cd Instantiator}
\DoxyCodeLine{\$ curl -\/s https://getcomposer.org/installer | php}
\DoxyCodeLine{\$ php composer.phar update}

\end{DoxyCode}
\hypertarget{md__c__workspace__proyecto__p_h_p_project_vendor_doctrine_instantiator__c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md7}{}\doxysection{Testing}\label{md__c__workspace__proyecto__p_h_p_project_vendor_doctrine_instantiator__c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md7}
The \mbox{\hyperlink{namespace_p_h_p_unit}{PHPUnit}} version to be used is the one installed as a dev-\/ dependency via composer\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\$ ./vendor/bin/phpunit}

\end{DoxyCode}
