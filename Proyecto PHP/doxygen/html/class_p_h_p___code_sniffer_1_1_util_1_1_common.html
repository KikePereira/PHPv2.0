static

Returns true if the specified string is in the camel caps format.

Parameters
string$stringThe string the verify.
boolean$classFormatIf true, check to see if the string is in the class format. Class format strings must start with a capital letter and contain no underscores.
boolean$publicIf true, the first character in the string must be an a-z character. If false, the character must be an underscore. This argument is only applicable if $classFormat is false.
boolean$strictIf true, the string must not have two capital letters next to each other. If false, a relaxed camel caps policy is used to allow for acronyms.
Returns
boolean
